# Aplicação de Reconhecimento Facial

## Descrição
Esta aplicação realiza o reconhecimento facial utilizando a biblioteca dlib e captura de vídeo com OpenCV. Os usuários podem registrar novos rostos e reconhecê-los em tempo real.

## Pré-requisitos
Antes de executar a aplicação, certifique-se de ter o Python instalado em sua máquina. A aplicação foi testada com Python 3.6 ou superior.
